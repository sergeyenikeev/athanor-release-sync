# Модель команды (обезличенная, демо)

| Роль | Зона ответственности | Каналы |
|---|---|---|
| Тимлид | координация релизов, решения по срокам | синк, почта |
| Владелец продукта | приоритеты, приёмка | синк, почта |
| Разработчик backend | сервис оплат (APP-4xx) | трекер, PR |
| Разработчик frontend | интеграции (APP-5xx) | трекер, PR |
| SRE | стенды, деплой, инциденты (OPS-xx) | трекер, почта |

Правила маршрутизации поручений:
- деплой/стенды/инциденты → SRE;
- задачи APP-4xx → Разработчик backend; APP-5xx → Разработчик frontend;
- приоритеты и перенос сроков → Владелец продукта (решение — Тимлид).
