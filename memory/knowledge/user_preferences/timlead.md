# Предпочтения пользователя (демо)

Целевой пользователь — тимлид по релизам (4–7 координационных встреч в день).

- Сводка — короткая, важное сверху: блокеры и конфликты — первыми.
  (Обратная связь от 03.07 → навык v2, см. skills/release_sync/versions/registry.json.)
- У каждого утверждения — источник и уровень уверенности.
- Конфликты источников — показывать оба значения, решение за человеком.
- Без воды и оценочных эпитетов: цифры и факты.
- Напоминания о блокерах — прямые, с владельцем и сроком.
