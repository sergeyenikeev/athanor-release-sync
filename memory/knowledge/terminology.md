# Терминология (единые термины во всех модулях и материалах)

| Термин | Определение |
|---|---|
| Релиз-синк | Координационная встреча перед релизом |
| Разбор инцидента | Встреча по разбору сбоя/инцидента |
| Брифинг/сводка | Подготовленный агентом контекст перед встречей |
| Поручение | action + owner + due + source |
| Решение | Зафиксированная договорённость (≠ идея) |
| Блокер | Препятствие релизу/инциденту |
| HITL | Human-in-the-loop — гейт подтверждения человека |
| MCP | Model Context Protocol — подключение внешних источников |
| Память релиза | Аудируемый markdown + журнал (memory/knowledge/release_*.md) |
| Навык (SKILL.md) | Повторяемый рабочий процесс агента Ouroboros |
| Evidence | Доказательство утверждения: источник + уверенность + выдержка |
