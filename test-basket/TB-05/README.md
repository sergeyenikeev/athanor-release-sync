# TB-05 · Поручение без срока

**task4:** СЦ-04 · **сложность:** low · **формат:** v1

Расшифровка: «Подготовь отчёт по инциденту». Срок не указан.

## Проверяемые функции

- извлечение поручений
- маркировка отсутствующего срока

## Проверяемые критерии

- нет выдуманного срока
- запрос уточнения
- due=не указан

## Что проверяется (checks)

- срок помечен «не указан», не выдумывается

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)

## Ожидаемое поведение

корректный (не выдуманный) срок.

## Возможный отказ

выдуманный срок

## Источники

Обязательные: calendar, tracker, transcript.
