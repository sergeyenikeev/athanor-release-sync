# TB-12 · Повторный прогон после обратной связи (формат v2)

**task4:** СЦ-15 · **сложность:** medium · **формат:** v2

Сводка слишком длинная; пользователь даёт обратную связь; применяется v2 (блокеры сверху, статусы ≤3) без деградации; откат.

## Проверяемые функции

- обратная связь
- эволюция навыка
- контрольные тесты
- откат

## Проверяемые критерии

- новая версия навыка
- нет деградации
- возможность отката

## Что проверяется (checks)

- формат v2: блокеры/конфликты сверху, статусы ≤3
- метрики не деградировали vs v1
- продемонстрирован rollback

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)

## Ожидаемое поведение

деградация метрик или потеря отката.

## Возможный отказ

деградация метрик или потеря отката
