**task4:** СЦ-01 · **сложность:** low · **формат:** v1

Все источники доступны, статусы не противоречат, есть задача и PR, одно обязательство с прошлого синка.

## Проверяемые функции

- определение встречи
- сбор контекста
- сводка
- извлечение решений/поручений
- память

## Проверяемые критерии

- end-to-end
- сводка
- источники
- память

## Что проверяется (checks)

- сводка: статусы задач и PR
- обязательства с прошлого синка
- 1 решение + 1 поручение

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)
