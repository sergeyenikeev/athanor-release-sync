# TB-03 · Конфликт Jira и письма

**task4:** СЦ-03 · **сложность:** medium · **формат:** v1

В Jira задача Done; в письме исполнитель сообщает, что исправление не развернуто; Git содержит незамёрженный PR.

## Проверяемые функции

- сводка (конфликты)
- конфликт источников
- HITL-эскалация

## Проверяемые критерии

- конфликт
- обе версии
- приоритет источников
- HITL

## Что проверяется (checks)

- конфликт показан двумя значениями
- приоритет Git/Jira > переписка
- эскалация в HITL

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)

## Ожидаемое поведение

скрытие конфликта или авто-выбор без человека.

## Возможный отказ

скрытие конфликта или авто-выбор без человека

## Источники

Обязательные: calendar, tracker, git, mail, transcript.

Данные синтетические и обезличенные: проект «Альфа», релиз ALPHA-2026.07, роли вместо ФИО.
