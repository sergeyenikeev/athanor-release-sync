# TB-04 · Расшифровка: решения и поручения

**task4:** доп. (глубокое извлечение) · **сложность:** medium · **формат:** v1

2 решения, 3 поручения (адресованное/самообязательство/безадресное), 2 идеи-дистрактора игнорируются, 1 блокер из расшифровки.

## Проверяемые функции

- извлечение решений
- извлечение поручений
- решение≠идея
- владелец/срок/источник
- извлечение блокеров из расшифровки

## Проверяемые критерии

- precision
- recall
- идеи-дистракторы
- владелец/срок
- блокеры

## Что проверяется (checks)

- 2 решения
- 3 поручения (адресованное/самообязательство/безадресное)
- 2 идеи-дистрактора игнорируются
- 1 блокер из расшифровки (OPS-77, ответственный SRE) — фиксация решением
  «блокер по OPS-77 фиксируем» попадает и в решения, и в структурированные блокеры

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)

## Ожидаемое поведение

выдуманное поручение или не пропускать решения.

## Возможный отказ

выдуманное поручение или пропуск решения

## Источники

Обязательные: calendar, tracker, transcript. Опциональные: mail.

Данные синтетические и обезличенные: проект «Альфа», релиз ALPHA-2026.07, роли вместо ФИО.
