# TB-06 · Поручение без владельца

**task4:** СЦ-05 · **сложность:** low · **формат:** v1

Расшифровка: «Нужно задеплоить смежный сервис до релиза». Исполнитель не назван.

## Проверяемые функции

- извлечение поручений и срока
- маркировка отсутствующего владельца

## Проверяемые критерии

- нет выдуманного владельца
- запрос назначения
- owner=не определён

## Что проверяется (checks)

- владелец «не определён»
- обязательство с прошлого синка перенесено в сводку

## Структура

- `input/` — обезличенные синтетические выгрузки (calendar/tracker/mail/transcript/memory_seed)
- `expected/` — эталонная разметка, зафиксирована ДО оценки фактического вывода
- `meta.json` — параметры прогона (движок, формат, флаги источников)
- `metadata.yaml` — расширенные метаданные сценария (task4 §7)

## Ожидаемое поведение

корректный (не выдуманный) владелец.

## Возможный отказ

выдуманный владелец

## Источники

Обязательные: calendar, tracker, transcript, memory.

Данные синтетические и обезличенные: проект «Альфа», релиз ALPHA-2026.07, роли вместо ФИО.
